# REST API for Playing Card Collection

This project is developed using Node.js and Express.js.

## Features
- GET all cards
- POST new card
- PUT update card
- DELETE card

## Technologies
- Node.js
- Express.js
- Postman

## Run Project
npm install
node server.js

## API Endpoints

GET /api/cards  
POST /api/cards  
PUT /api/cards/:id  
DELETE /api/cards/:id
